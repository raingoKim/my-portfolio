# 김정윤 포트폴리오

현대적이고 전문적인 포트폴리오 랜딩 페이지입니다.

## 기술 스택

- React 18
- Vite
- Tailwind CSS
- Framer Motion (애니메이션)

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 주요 기능

- 다크 모드 디자인
- 반응형 레이아웃
- 스크롤 페이드인 애니메이션
- 인터랙티브 배경 애니메이션
- 경력 타임라인
- 자격증 및 기술 스택 시각화
